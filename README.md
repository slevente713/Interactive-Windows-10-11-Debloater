# Interactive-Windows-10/11-Debloater

You can uninstall almost every app that can be pre-installed on any Windows 8/8.1/10/11 machine. This script don't have a GUI yet, you need to run this with Windows PowerShell or PowerShell 7 (recommended)

Le tudsz vele törölni majdnem minden programot ami előretelepülhet egy Windows 8/8.1/10/11 gépre. Ennek a szkriptnek még nincs grafikus felülete, Windows PowerShell-lel / PowerShell 7-tel (javasolt) kell futtatni.

How to use? Start Windows PowerShell or PowerShell 7 (recommended) as administrator. Then type this: "Set-ExecutionPolicy -ExecutionPolicy Bypass -Scope LocalMachine -Force", and then press enter. Then, type this: "Set-ExecutionPolicy -ExecutionPolicy Bypass -Scope CurrentUser -Force", then press enter. Then change the directory to the folder where you downloaded my script (like if you downloaded to the Download folder then type this: "cd C:\Users\Your Username\Downloads", and then press enter again). Then type this: "./Debloater.ps1/", and answer the questions. If you finished, then open another admin PowerShell, and type these: "Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope LocalMachine -Force", and "Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser -Force".

Hogyan használd? Indítsd el a Windows PowerShell-t vagy a PowerShell 7-et (javasolt) rendszergazdaként. Aztán írd be ezt: "Set-ExecutionPolicy -ExecutionPolicy Bypass -Scope LocalMachine -Force", majd nyomd meg az entert. Most írd be ezt: "Set-ExecutionPolicy -ExecutionPolicy Bypass -Scope CurrentUser -Force", és megint nyomj egy entert. Most módosítani kell a jelenlegi directory-t, oda ahová letoltötted a szkriptem (pl: Ha a letöltések mappába töltötted le, akkor írd be ezt: "cd C:\Users\Felhasználóneved\Downloads", és nyomj entert). Aztán írd be ezt: "./Debloater.ps1/", és válaszold meg a kérdéseket. Ha végeztél akkor nyiss egy új rendszergazda PowerShell-t, és írd be ezeket: "Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope LocalMachine -Force", és "Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser -Force".
